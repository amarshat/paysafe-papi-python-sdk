The Paysafe Python SDK provides a clean, modern interface to the Paysafe API. It handles authentication, request formatting, and response parsing, allowing you to focus on your business logic.

Quick Example

from paysafe import Client
from paysafe.models.payment import Payment, CardPaymentMethod

# Initialize the client
client = Client(api_key="your_api_key", environment="sandbox")

# Create a payment
payment = Payment(
    amount=1000,  # $10.00
    currency_code="USD",
    payment_method=CardPaymentMethod(
        card_number="4111111111111111",
        card_expiry={"month": 12, "year": 25},
        card_holder_name="John Doe",
        card_cvv="123"
    ),
    description="Test payment"
)

# Submit the payment
result = client.Payment.create(payment)
print(f"Payment created: {result.id}")
